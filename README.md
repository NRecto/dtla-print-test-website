# DTLA Print Landing Page

> **Note: This is a test website**

A modern and simple landing page for DTLA Print, showcasing their high-quality custom merchandise services including embroidery, eco-friendly screen printing, and full package finishing services.

## About the Website

At DTLA Print, we believe **Not All Prints Are Created™**. We are a High Quality Boutique Style one stop shop for high quality custom merch. We offer Embroidery, eco-friendly Screen Printing and Full Package Finishing Services located in Sunny Los Angeles, California.

We take the brands you love and embroider your logo on the material. Whether it's a hoodie, hat, custom t-shirt or a bag - we produce everything in-house to be able to offer low wholesale prices.

### Features

- **Home Section**: Hero section with tagline and call-to-action buttons
- **About Section**: Company information and social media links
- **Waitlist Form**: Email signup form for early access
- **Contact Form**: Contact form for inquiries and appointments
- **Responsive Design**: Mobile-friendly navigation and layout
- **Smooth Scrolling**: Seamless navigation between sections

## Tech Stack

### Dependencies

- **Next.js** (^14.2.0) - React framework for production
- **React** (^18.3.0) - JavaScript library for building user interfaces
- **React DOM** (^18.3.0) - React package for working with the DOM

### Dev Dependencies

- **Tailwind CSS** (^3.4.3) - Utility-first CSS framework
- **PostCSS** (^8.4.38) - CSS transformation tool
- **Autoprefixer** (^10.4.19) - PostCSS plugin to parse CSS and add vendor prefixes
- **ESLint** (^8.57.0) - JavaScript linter
- **eslint-config-next** (^14.2.0) - ESLint configuration for Next.js

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint
